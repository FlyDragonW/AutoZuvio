# AutoZuvio
自動檢查 Zuvio 上是否有新題目  

Disclaimer: 僅供找不到 Ubike 停車位時使用，請不要翹課。  

## TL;DR
在 `main.py` 填入 credentials，執行腳本即可在有新題目時發出鈴聲提醒。

## Usage
1. clone the repo
```
git clone https://github.com/FlyDragonW/AutoZuvio.git
cd AutoZuvio
```
2. install requirements
```
pip install -r requirements.txt
```
3. Fill in info
- Email (Account)  
- Password
- target url
4. Enjoy :)
```
python3 main.py
```
